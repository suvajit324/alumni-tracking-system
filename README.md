# 🎓 Alumni Tracking System

> 🚀 End-to-End Data Engineering Project with Interactive Dashboard

---

## 🧠 Problem Statement
Many institutions lack a centralized system to track alumni career data such as job roles, salary trends, and locations. This leads to inefficient analysis and poor visibility of alumni outcomes.

---

## 💡 Solution
This project builds a complete data pipeline that:
✔ Collects alumni data (CSV)  
✔ Cleans and transforms data (Python - ETL)  
✔ Stores data in a structured database (SQLite)  
✔ Visualizes insights via an interactive dashboard (Streamlit)  

---

## 🚀 Features

📥 Data Ingestion from CSV  
🧹 Data Cleaning & Transformation  
🗄️ SQLite Database Storage  
📊 Interactive Dashboard (Streamlit)  
🔍 Filters (Department, Year, Salary)  
🔎 Search Alumni by Name  
📈 Charts & Metrics  
⬇ Download Data Option  

---

## 🛠️ Tech Stack

Language      : Python  
Libraries     : Pandas, NumPy  
Database      : SQLite  
Frontend      : Streamlit  
Query Language: SQL  

---

## 🔄 Workflow

CSV → Python (ETL) → SQLite Database → Streamlit Dashboard

---

## ▶️ How to Run

### 1️⃣ Install Dependencies
pip install pandas numpy sqlalchemy streamlit

### 2️⃣ Generate Dataset
python scripts/generate_data.py

### 3️⃣ Run ETL Pipeline
python scripts/transformation.py  
python scripts/load.py  

### 4️⃣ Launch Dashboard
python -m streamlit run app.py  

Open in browser:  
https://alumni-tracking-system-fieqyfwycdkbspycfmk2mn.streamlit.app/
